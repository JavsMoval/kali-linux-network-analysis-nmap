# Kali Linux Network Analysis with Nmap

This project presents a technical report on network scanning using Nmap in a Kali Linux environment.

## Objectives
- Perform host discovery
- Scan open ports
- Identify running services
- Analyze network security

## Tools Used
- Kali Linux
- Nmap
- PuTTY

## Project Structure
- /report → Technical report document
- /screenshots → Evidence of scans performed
